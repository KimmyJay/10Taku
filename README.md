## 1. 프로젝트 주제
### 애니메이션 추천
```
AI가 사용자에게 애니메이션을 추천해주는 프로그램입니다.  
처음 애니메이션에 입문한 분들 혹은 이미 좋아하시는 분들을 위한 애니메이션 추천 사이트입니다.
```

## 2. 머선12조 팀원 및 역할분담

    * 발표
